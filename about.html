Project Description

For this project, I chose to work alone on a Fictional Company's Website. My fictional company is called 'The Closet' and has a clothing store in Kalamazoo. This website has 7 viewable pages.

My website's homepage (index.php) has a bootstrap carousel, tells visitors about this store and gives them two links to contact the store: phone and email. The page also shows store's timings and a Google Map centered at Kalamazoo since the store's address is fictional.

'Our products' page (catalog.php) lists (using <dl>) the items sold at the store along with descriptions and costs. All the displayed content is obtained directly from database using a PDO object and SQL code. Animations and events have also been used to magnify images as users hover over them.

'Deals' page (deals.html) uses a table to list the promotions currently being offered by the store. It also uses animations to animate entries in the first column.

'Rate Us' page uses a form that allows visitors to rate 'The Closet' store. For comments, I've limited the number of characters to 280 and have added a character count underneath the textbox which changes with every key press and changes color to red when user has 50 or less characters left. The form's action attribute leads to a PHP page (ratingSubmit.php) which stores the user's responses in database and notifies the user accordingly.

'For Staff' page allows employees of the store to check inventory status of individual products. To do so, employees key in their ID and Password. I'm using 'keyup' event to monitor inputs and so no button needs to clicked to log in; as soon as the user keys in the right combination, he/she is greeted with the name associated with the entered id and is shown another text box in which he/she can enter a product's title to know its 'Stock Status'. I've also given users the option to change their passwords and am using md5 and unique seeds to store them. All of the prior three are achieved using AJAX and so the page never reloads or redirects. I've also utilized jQuery's sliding animations on this page.

And, this page (about.html) tells about me and this project. It also contains an embedded SoundCloud audio and an embedded YouTube video.

I decided to upload my project to 'AwardSpace'. My website's link is http://thecloset.myartsonline.com/.
